read 的参数

  • -t ： 读取输入时设置超时秒数，也可以使用环境变量 TMOUT 来指定超时秒数
  • -p ：指定输入的提示信息
  • -n ：指定只读取若干个字符：
  • -e ：允许使用 readline 库提供的快捷键，例如自动补全，在输入路径时很有用：
  • -s ：用户输入不显示出来，适合输入密码

其他不太常用的参数：

  • -a ：将用户的输入保存为一个数组：
  • -d (delimiter) ：指定输入结束符，默认时换行结束
  • -r ：raw 模式，不会转义输入的反斜杠转义字符
  • -u fd ：使用文件描述符 fd 作为输入

IFS 变量

IFS，Internal Field Separator，内部字段分隔符，默认值为空格、Tab、换行符。
