子字符串

截取子字符串：

  • offset: 其实索引，从 0 开始，如果为负值则表示从结尾开始，注意负值必须在负号前增加空格，否则和${name:-value} 冲突，负值正确写法：${name: -6}
  • length: 截取的长度，不指定则截取到末尾，length 可以为负，如果为负，则表示排除末尾的字符，如 -2 表示排除结尾 2 个字符
